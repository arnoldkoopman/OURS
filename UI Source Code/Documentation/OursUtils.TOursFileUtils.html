Constructor will raise exception as creation of an instance of this class is not allowed.

Class only contains class functions.

class function DocumentsDir: string;

Returns the documents folder (e.g. C:\Users\<user>\Documents\OURS).

class function CommonDocumentsDir: string;

Returns the documents folder for all users (e.g. C:\Users\Public\Public Documents\OURS).

class function TempDir: string;

Returns the folder for temporary files.

This folder is a subfolder of the temp-folder. The subfolder is 'OURS_' followed by the process-id.

class function PythonDir: string;

Returns the folder where Python is located.

This folder is the subfolder 'Python' in the program folder.
