procedure ShowUsage(str: string; bHalt: Boolean = True);

Displays message with correct usage of the software.

Behaviour depends on the number of command line options:
0: The software is started in test-mode with full user-interface and OURS_input.xml is used as inputfile.

1: The software is started in test-mode with full user-interface and <parameter1> is used as inputfile.

2: <parameter1> is used as inputfile, calculation is being executed and the results are written to <parameter2>.
Only the progress dialogue with progress and messages will be visible. Software will close when calculation is completed.

3 or more: Same as 2 parameters, but with additional options (can be combined):
- '/TEST' : The software is started in test-mode and detailed results will be written to a TXT-file.
- '/SILENT' : The software is started in silent mode. No user-inteface will be visible and the software is closed when the calculation is completed.
