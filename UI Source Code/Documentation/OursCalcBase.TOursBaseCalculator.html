type TOursBaseCalculator = class(TObject)

Description

This is the base class for all Python calculators.

Use the class functions to retrieve the corresponding calculators.

FProcessInfo: TProcessInformation;

Enables access to the process handle and thread for terminating the calculation.

Exitcode in case the calculation is terminated by the user.

Exitcode in case the script does not produce results. Probably due to an unsuspected error in the script.

Methods

function AddChildProcesses(aList: TList<DWORD>; aPID: DWORD): Boolean;

Adds all child processes of aPID to aList.

procedure EndModule(aPID: DWORD = 0);

Ends a process with aPID.

Running Python creates an unknown number of child processes. To successfully terminate a calculation all child processes need to be identified and terminated seperately.

function RunModule(progStr, params, outputFile: string): Cardinal;

Starts the process "progStr" with parameters "params" and waits until the process is ended.

The function returns the exitcode of the process. 0 indicates success.

function TempAvailable: Boolean;

Indicates if the OURS folder for temporary files is present and writable.

The OURS folder for temporary files is a folder located in the Windows temp-folder. The name of this folder is 'OURS_' followed by the process-id.

function GetModuleName: string; virtual; abstract;

Abstract function to retrieve the module name (filename of the Python script).

function SaveInFile(inputJSON: string; out inFile: string): Boolean; virtual;

Save the JSON string to a file and returns success.

If the data is successfully saved "inFile" contains the name of the file.
The file is placed in the program tempfolder and is called "input.json"

function OutFolderAvailable(out outFolder: string): Boolean; virtual;

Determines the name of the output folder and returns if this folder exists.

The output folder is located in the program tempfolder and is "OUT".
If the folder already exists, any files in it will be deleted.

function GetParams(const inStr, outStr: string): string; virtual;

Returns the command line parameters which will start the Python script.

Python prefers '/' as path delimeter and not '\': all backslashes will be replaced.

function ProgAvailable(out progStr: string): Boolean; virtual;

Checks if python.exe and the python script (modulename) exist.

python.exe and the Python script given by ModuleName need to be located in the folder Python which is located in the OURS program folder.
