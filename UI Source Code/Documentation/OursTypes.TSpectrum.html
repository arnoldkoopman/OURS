Private procedure to set the value for a given frequency band

constructor Create;

Constructor: depending on the configuration it will create a spectrum for 1/1/-octave or 1/3-octave.

destructor Destroy; override;

Destructor.

function AsText: string;

Returns the contents as text.

As decimal separator the user defined setting is used and the values are seperated by a semicolon.

function AsJsonText: string;

Returns the contents as JSON text.

As decimal separator '.' is used and the values are seperated by a comma and a space. The values are enclosed between brackets.

procedure FillValue(const Value: Double);

Fills each frequency band with the given value.

procedure FillValues(const Value: TSpectrum);

Copies the values for each frequency band from the given spectrum.

procedure FillFromArray(AnArray: TArray<Extended>);

Creates an instance of TSpectrum and fills it with the values from AnArray.

If AnArray has more or less values than TSpectrum, the results in nil.

procedure AssignFromArray(AnArray: TArray<Extended>);

Copies the values from AnArray.

If AnArray has less values the resulting values will be set to zero.
