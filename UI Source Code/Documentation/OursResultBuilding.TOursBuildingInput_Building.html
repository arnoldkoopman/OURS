type TOursBuildingInput_Building = class(TObject)

Description

Class for storing the section "Gebouw" for the the Building module.

These values are stored in TOursReceptor (constructionyear, apartment, numberoffloors, walllength, facadelength). Values of var_* are currently set to zero.
