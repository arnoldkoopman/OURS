procedure FormCreate(Sender: TObject);

Event handler for the constructor of the mainform.

Method will setup the mainform, open the database, display table contents and process the command line parameters.

procedure FormDestroy(Sender: TObject);

Event handler for the destructor of the mainform.

Method will free all private data.

procedure FormActivate(Sender: TObject);

Event handler for activating the mainform.

Depending on the command line options the calculation will be started and/or the full user-interface will be visible.
This event handler will only be called once.

Event handler for resizing the mainform.

This method will distribute the tabs evenly depanding on the width of the mainform.
