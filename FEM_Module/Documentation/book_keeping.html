book_keeping module¶

This module includes functions that take care of the bookkeeping. Bookkeeping consists of assigning equation numbers to degrees of freedom.

book_keeping.connectivity(elements, node_id)¶

Determine the equation numbers of each dof in each element (i.e. connectivity matrix)

Parameters

  • elements – (int) array [NE, 4] with the node numbers of each finite element corner (NE: total number of elements)

  • node_id – (int) array [NP, 2] with the equation number of the radial and vertical displacement of each node (NP: total number of nodes)

Returns

(int) array [NE, 8] equation numbers of each dof in each element

book_keeping.mapping(nodes, bounds)¶

Assign each degree of freedom of each node to an equation number

Parameters

  • nodes – (float) array [NP, 2] with the radial and vertical coordinate of each node (NP: total number of nodes)

  • bounds –

    (int) parameter to determine the set of boundary conditions

    • bounds == 0: left = symmetric, bottom = fixed, right = free

    • bounds == 1: left = symmetric, bottom = fixed, right = infinite

    • bounds == 2: left = symmetric, bottom = infinite, right = infinite

    • bounds == 3: left = symmetric, bottom = infinite, right = infinite

Returns

(int) array [NP, 2] equation numbers of the radial and vertical displacement of each node
