main module¶

This constitutes the main module, in which the input data is read, the solution is generated and where it is finally exported into an output file

main.axifem2(file_in, file_out, write_file)¶

Main function that reads the input, sets up the FEM model, performs the simulation and write the results

Parameters

  • file_in – (string) full path to the json-file describing the input for the FEM model

  • file_out – (string) full path to the json-file with the output of the FEM calculations

  • write_file – (string) full path to the txt-file describing the determined FEM parameters

(void)
